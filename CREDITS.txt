Kokoro TTS v1.0
  License : Apache 2.0
  Source  : https://github.com/hexgrad/kokoro
  Notes   : Fast ONNX-based neural TTS engine. Apache 2.0 permits commercial
            use with attribution.

kokoro-onnx (Python bindings)
  License : Apache 2.0
  Source  : https://github.com/thewh1teagle/kokoro-onnx

Chatterbox TTS
  License : MIT
  Source  : https://github.com/ResembleAI/chatterbox
  Notes   : Natural-sounding neural TTS by Resemble AI. Natural mode output
            includes an inaudible AI watermark via the Perth library (see below).

Perth Watermarker
  License : MIT
  Source  : https://github.com/resemble-ai/perth
  Notes   : Imperceptible neural watermark embedded in all Chatterbox output.
